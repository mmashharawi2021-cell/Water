# Water

نظام تقارير تشغيل وضخ المياه.

## الهدف

إدارة التقارير اليومية، الوقود، المياه، الجهات المستفيدة، والتصدير من واجهة واحدة.

## قواعد النسخة الجديدة

- لا ملفات patch عشوائية.
- لا نصوص تسويقية زائدة داخل الواجهة.
- الوقود الوارد من سجل مستقل.
- الوقود المستخدم من التقارير اليومية.
- الوقود المتبقي = الوارد - المستخدم للفترة المعتمدة.
